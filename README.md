# varnish

[![Build](https://github.com/gsmlg-ci/varnish/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/varnish/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/varnish`
- `ghcr.io/gsmlg-dev/varnish`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/varnish:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/varnish:latest
```

## Build

```bash
docker build -t gsmlg/varnish .
```
